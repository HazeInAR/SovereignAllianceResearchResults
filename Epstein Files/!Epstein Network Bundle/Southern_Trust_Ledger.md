- **Flight manifests / maintenance**: The N212JE manifests (EFTAO0133526, EFTAO0133529, EFTAO01267394/87, HOUSE_OVERSIGHT_005290) list Jeffrey Epstein as passenger #1 and detail the flights flown between St. Thomas, Teterboro, West Palm Beach, Paris, Doha, etc. Every manifest page identifies the flight desk as Plan D c/o Atlantic / attn: Larry Visoski, meaning the Southern Trust wires ultimately paid for those charters. These are the documents that connect each Southern Trust inflow with a specific set of Plan D flights.
