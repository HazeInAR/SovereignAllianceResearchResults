## Visual guidance
- Plot the Jericho payments on the left axis (May–Aug 2013), mark amounts ($250K, $65K, $157K) and tie them to the JEGE1085 label as a single sprint of aviation spending. Use the same color for the JEGE tag so the line stands out.
- Plot the Delray transfers as persistent horizontal bands across the timeline (2001 → 2008 → 2013 → 2017 → 2019). Note the handler triad (Indyke/Kahn/Oldfield) and RC addresses (River Vale, Nanuet, Wellington, 5300 W Atlantic Ave Suite 602). When the 2013 Delray rotation happens, highlight that it coincides with the Jericho payment cluster.
- Add callouts explaining that the same nominee managers (Basia McDonnell + Ornit Levinson-Suter) appear on both the condo deeds and the moving-company invoices that funded the JEGE wires. That makes the timeline a single node linking aviation payments to property control.
