# Republican Jewish Coalition PAC — Conservative/Israel Influence Dossier

This dossier captures the overlapping donors, the recipient totals by cycle, and the affiliate activity so we can treat RJC as a cultural-lobby node inside the broader `#AIPAC` / leadership pipeline.
