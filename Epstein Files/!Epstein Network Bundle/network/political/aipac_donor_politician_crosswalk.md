2. **Politician tags:** Each politician is already fed into the `Foreign Agent Politicians` section (Lindsey Graham, Mike Johnson, Ted Cruz, etc.). Add the donor tags there when you discover new contributions from this ledger.
3. **Narrative anchors:** Link every donor-politician pair to one of the Israel timeline beats (Abraham Accords/normalization, Gaza/Hamas escalation, Hezbollah proxy spikes, etc.) so the crosswalk doubles as a storyline map.
4. **Verification:** If you pull any underlying FEC or OpenSecrets data, append it under the `Sources` heading for each row so reviewers can trace back to the original PAC expenditures or IE filings.
